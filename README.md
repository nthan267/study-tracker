# Study Tracker

A full-stack web application to log, review, and manage personal study sessions across coursework. Built with TypeScript, React, Node, Express, and MongoDB.

**Live demo:** https://study-tracker-one-blond.vercel.app

## Stack

- **Frontend:** React, TypeScript, Vite
- **Backend:** Node.js, Express, TypeScript
- **Database:** MongoDB Atlas
- **Hosting:** Vercel (frontend), Render (backend)

## Features

- Add study sessions with course, duration, topics, notes, and productivity rating
- List all past sessions
- Delete sessions
- Data persists in MongoDB Atlas

## Roadmap

Currently a session log. Evolving into a full course-aware study tracker with:

- Course management (add courses per semester)
- Task types: assignments, labs, study sessions, projects
- Completion tracking and % progress per course
- Upcoming deadlines view
